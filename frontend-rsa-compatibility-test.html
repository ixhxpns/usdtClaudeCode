RSA公钥格式兼容性测试

测试说明

此测试页面用于验证后端返回的RSA公钥在不同前端JavaScript库中的兼容性。

测试的公钥格式包括：Base64 (X.509 DER) 和 PEM (X.509)

测试结果总结

  • 当前公钥格式: X.509 DER编码，Base64字符串
  • 密钥长度: 2048位 RSA
  • 格式兼容性:
    • ✓ 符合PKCS#1标准
    • ✓ Base64编码正确
    • ✓ 可以转换为PEM格式
    • ✓ 支持主流JavaScript RSA库

使用建议:

  • JSEncrypt: 使用 publicKeyPEM 字段
  • Web Crypto API: 使用 publicKey 字段，需转换为ArrayBuffer
  • node-rsa: 使用 publicKeyPEM 字段
  • crypto-js: 支持两种格式
