## 系統架構概覽
- **管理員前端**: Vue 3 + Pinia + Element Plus + Vite
- **用戶前端**: Vue 3 + Pinia + Element Plus + Vite + ECharts + RSA加密
- **後端服務**: Spring Boot + MyBatis + JWT + MySQL + Redis
- **協議**: HTTPS + RESTful API + Swagger/OpenAPI
